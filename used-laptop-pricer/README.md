# Used Laptop Pricer - افزونه تخمین قیمت لپ‌تاپ دست دوم

## 📋 توضیحات
افزونه وردپرس برای تخمین قیمت لپ‌تاپ‌های دست دوم با انتخاب مدل قطعات مختلف. این افزونه امکان محاسبه قیمت را بدون بارگذاری مجدد صفحه فراهم می‌کند.

## ✨ ویژگی‌ها
- ✅ محاسبه قیمت با AJAX (بدون رفرش صفحه)
- ✅ مدیریت مدل‌های قطعات در پنل مدیریت
- ✅ پشتیبانی از زبان فارسی و RTL
- ✅ طراحی ریسپانسیو
- ✅ نمایش جزئیات قیمت هر قطعه
- ✅ اعمال ضریب استهلاک برای لپ‌تاپ‌های دست دوم
- ✅ فرم‌های اعتبارسنجی شده
- ✅ رابط کاربری مدرن و زیبا

## 🚀 نصب و راه‌اندازی

### روش ۱: نصب از طریق ZIP
1. فایل ZIP افزونه را دانلود کنید
2. در پنل مدیریت وردپرس به بخش "افزونه‌ها" بروید
3. روی "افزودن" کلیک کنید
4. فایل ZIP را آپلود و نصب کنید
5. افزونه را فعال کنید

### روش ۲: نصب دستی
1. فایل‌های افزونه را در پوشه `/wp-content/plugins/used-laptop-pricer/` کپی کنید
2. از پنل مدیریت وردپرس افزونه را فعال کنید

## ⚙️ تنظیمات اولیه

### ۱. تنظیم مدل‌های قطعات
1. به پنل مدیریت وردپرس بروید
2. از منوی "لپ‌تاپ پرایسر" استفاده کنید
3. برای هر دسته قطعات (CPU، RAM، GPU، Storage) مدل‌ها و قیمت‌ها را وارد کنید

### ۲. استفاده در سایت
از شورت‌کد زیر در صفحات یا پست‌ها استفاده کنید:
```
[used_laptop_pricer]
```


## 🎯 نحوه استفاده

### برای مدیران سایت:
1. **مدیریت قطعات**: در پنل مدیریت، مدل‌های مختلف CPU، RAM، GPU و Storage را با قیمت‌های مربوطه وارد کنید
2. **حذف و ویرایش**: امکان حذف یا ویرایش هر مدل از طریق دکمه‌های مربوطه

### برای کاربران:
1. **انتخاب برند و مدل**: نام برند و مدل لپ‌تاپ را وارد کنید
2. **انتخاب قطعات**: از منوهای کشویی، مدل‌های مختلف قطعات را انتخاب کنید
3. **محاسبه قیمت**: روی دکمه "محاسبه قیمت" کلیک کنید
4. **مشاهده نتیجه**: قیمت نهایی با جزئیات هر قطعه نمایش داده می‌شود

## 🔧 تنظیمات پیشرفته

### تغییر ضریب استهلاک
در فایل `includes/calculate-price.php` خط ۴۷ را ویرایش کنید:
```php
$depreciation_factor = 0.7; // 70% of new price
```

### شخصی‌سازی استایل‌ها
فایل `assets/css/style.css` را ویرایش کنید تا ظاهر فرم را تغییر دهید.

## 🛠️ عیب‌یابی

### مشکل: فرم نمایش داده نمی‌شود
- بررسی کنید که شورت‌کد `[used_laptop_pricer]` درست وارد شده باشد
- افزونه را غیرفعال و دوباره فعال کنید

### مشکل: قیمت محاسبه نمی‌شود
- بررسی کنید که مدل‌های قطعات در پنل مدیریت وارد شده باشند
- کنسول مرورگر را برای خطاهای JavaScript بررسی کنید

### مشکل: استایل‌ها اعمال نمی‌شوند
- کش مرورگر را پاک کنید
- بررسی کنید که فایل‌های CSS به درستی بارگذاری شوند
