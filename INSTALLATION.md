# راهنمای نصب افزونه Used Laptop Pricer

## 📦 فایل‌های آماده شده

✅ **فایل ZIP آماده نصب**: `used-laptop-pricer-v3.4.zip`

## 🚀 مراحل نصب

### مرحله ۱: نصب افزونه
1. فایل `used-laptop-pricer-v3.4.zip` را دانلود کنید
2. وارد پنل مدیریت وردپرس شوید
3. به بخش **افزونه‌ها** → **افزودن** بروید
4. روی **آپلود افزونه** کلیک کنید
5. فایل ZIP را انتخاب و آپلود کنید
6. روی **نصب** کلیک کنید
7. افزونه را **فعال** کنید

### مرحله ۲: تنظیم مدل‌های قطعات
1. از منوی اصلی، **لپ‌تاپ پرایسر** را انتخاب کنید
2. برای هر دسته قطعات، مدل‌ها و قیمت‌ها را وارد کنید:

#### مثال برای CPU:
- نام مدل: `i5-1135G7`
- قیمت: `2400000`

#### مثال برای RAM:
- نام مدل: `8GB DDR4`
- قیمت: `800000`

#### مثال برای GPU:
- نام مدل: `GTX 1650`
- قیمت: `1500000`

#### مثال برای Storage:
- نام مدل: `512GB SSD`
- قیمت: `1200000`

### مرحله ۳: استفاده در سایت
در هر صفحه یا پست، شورت‌کد زیر را قرار دهید:
```
[used_laptop_pricer]
```

## 🔧 تنظیمات پیشرفته

### تغییر ضریب استهلاک
در فایل `includes/calculate-price.php` خط ۴۷:
```php
$depreciation_factor = 0.7; // 70% of new price
```

### شخصی‌سازی ظاهر
فایل `assets/css/style.css` را ویرایش کنید.
